# RLagent
Lu Ban Training on the King Enlightenment Platform

修改的时候注明修改的文件和原参数（并将文件上传或创建一个相同名字的空白文件）


**任务分配：**

1.修改代码，调整模型等（3人）

2.实验报告+PPT（2人）



**任务要求：**

1.修改代码人员：（每周一汇报进度）

- 进行模型训练
- 下载训练好的模型（并分享给其他人）
- 查看训练日志和录像
- 分析存在的问题
- 修改参数并重复上面训练

2.报告、PPT人员：

- 在每周进度汇报时要参与，并记录修改内容（因为涉及到内容上的修改）
- 在15周之前完成PPT和实验报告的雏形。
